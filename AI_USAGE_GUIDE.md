# 🤖 ESP32-S3 智能摄像头 - AI功能使用指南

## 🎯 功能概述

本项目集成了多种AI智能功能，为用户提供自然语言交互、物体识别、智能搜索和自动驾驶等先进功能。

### 🧠 AI功能架构
```
AI智能系统
├── 🌐 云端AI服务 (GPT/Claude等)
│   ├── 自然语言命令处理
│   ├── 图像分析和描述
│   └── 智能对话交互
├── 🏠 本地AI服务
│   ├── 轻量级物体检测
│   ├── 颜色特征识别
│   └── 边缘检测分析
└── 🎮 任务控制系统
    ├── 智能搜索任务
    ├── 自动驾驶模式
    └── 实时状态监控
```

## 🎯 AI搜索任务系统

### 任务配置

#### 基本设置
```
🎯 目标物品: [输入要搜索的物品名称]
⏱️ 任务时间: 10-300秒 (可调节)
🔄 搜索模式: 静态/移动搜索
```

#### 高级设置
```
🎨 颜色过滤: 指定目标颜色
📏 尺寸过滤: 大/中/小物体
🔍 精度设置: 高精度/快速模式
```

### 任务执行流程

#### 1. 任务启动
```
用户输入 → 系统解析 → 任务创建 → 开始搜索
```

#### 2. 搜索过程
```
拍照 → 本地AI检测 → 云端AI确认 → 结果分析
  ↓
如未找到 → 机器人移动 → 重新拍照 → 继续搜索
```

#### 3. 任务状态
- `🟢 IDLE`: 空闲状态
- `🟡 SEARCHING`: 正在搜索
- `🔵 SCANNING`: 360度扫描
- `✅ COMPLETED`: 任务完成
- `❌ FAILED`: 任务失败

## 🏠 本地AI检测功能

### 支持的物体类别
基于COCO数据集的80种物体类别：

#### 常见物体
```
👥 人物: person
🚗 交通工具: car, bicycle, motorcycle, bus, truck
🐱 动物: cat, dog, bird, horse, cow, sheep
🍎 食物: apple, banana, orange, cake, pizza
📱 电子设备: laptop, mouse, keyboard, cell phone
🪑 家具: chair, couch, bed, dining table
```

#### 检测特性
- **实时检测**: 延迟100-500ms
- **置信度评估**: 0.0-1.0分数
- **边界框**: 精确定位物体位置
- **多物体**: 同时检测多个目标

### 颜色特征识别

#### 支持的颜色
```
🔴 红色物体检测
🟢 绿色物体检测
🔵 蓝色物体检测 (计划中)
🟡 黄色物体检测 (计划中)
```

#### 检测算法
```
HSV颜色空间分析 → 颜色阈值过滤 → 轮廓检测 → 特征提取
```

### 边缘检测功能
- **Canny边缘检测**: 识别物体轮廓
- **轮廓分析**: 计算面积和周长
- **形状识别**: 圆形、矩形、三角形等

## 🚗 AI自动驾驶模式

### 功能特性

#### 智能导航
- **障碍物检测**: 实时识别前方障碍
- **路径规划**: 自动选择最优路径
- **目标跟踪**: 跟随指定目标移动
- **边界识别**: 识别可通行区域

#### 安全机制
- **紧急停止**: 检测到危险立即停止
- **速度控制**: 根据环境自动调速

## 🔧 API接口详解

### 核心API端点

#### 1. AI命令执行
```http
POST /ai-command
Content-Type: application/json

{
  "command": "帮我找一下杯子",
  "timeout": 30,
  "mode": "search"
}

响应:
{
  "status": "success",
  "result": "发现白色杯子在桌子左侧",
  "execution_time": "2024-12-19 10:30:15",
  "confidence": 0.85
}
```

#### 2. AI任务控制
```http
POST /api/ai-task
Content-Type: application/json

{
  "target_object": "钥匙",
  "timeout_seconds": 60,
  "search_mode": "active"
}

响应:
{
  "task_id": "search_001",
  "status": "started",
  "message": "开始搜索钥匙"
}
```

#### 3. 任务状态查询
```http
GET /api/ai-task-status

响应:
{
  "task_id": "search_001",
  "status": "SEARCHING",
  "progress": 45,
  "message": "正在第3个位置搜索",
  "elapsed_time": 27,
  "remaining_time": 33
}
```

## 🛠️ 故障排除

### 常见问题

#### 1. AI命令无响应
```
可能原因:
- 网络连接问题
- AI服务配置错误
- 命令格式不正确

解决方案:
1. 检查网络连接状态
2. 验证AI服务配置
3. 尝试简单的命令
4. 查看串口调试信息
```

#### 2. 物体检测不准确
```
可能原因:
- 光线条件不佳
- 物体部分遮挡
- 摄像头焦距问题

解决方案:
1. 改善照明条件
2. 调整摄像头角度
3. 清洁摄像头镜头
4. 降低检测敏感度
```

#### 3. 搜索任务频繁失败
```
可能原因:
- 目标物体不在范围内
- 搜索时间设置过短
- 电机控制问题

解决方案:
1. 扩大搜索范围
2. 增加任务超时时间
3. 检查电机连接
4. 手动验证电机功能
```
