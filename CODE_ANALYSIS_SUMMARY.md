## 技术栈总结

### 硬件平台
- **主控芯片:** ESP32-S3 (双核Xtensa LX7，WiFi+蓝牙)
- **摄像头:** OV2640/OV5640 (JPEG输出)
- **电机驱动:** L298N双H桥驱动
- **存储:** 8MB Flash + 512KB SRAM
- **连接:** WiFi 2.4GHz, GPIO, SPI, I2C, UART

### 软件框架
- **操作系统:** FreeRTOS 10.x
- **开发框架:** ESP-IDF 5.2
- **编程语言:** C语言 (符合C89标准)
- **网络协议:** HTTP/HTTPS, WebSocket, TCP/IP
- **数据格式:** JSON, Base64, JPEG

### 第三方服务
- **云端AI:** Mistral AI API
- **时间同步:** NTP服务
- **证书验证:** ESP Certificate Bundle
- **JSON处理:** cJSON库
- **加密算法:** mbedTLS

## 性能指标

### 系统性能
- **启动时间:** ~10秒（包含WiFi连接）
- **图像采集:** 1-3秒/帧（根据模式）
- **本地AI检测:** <500ms
- **云端AI分析:** 2-5秒（依赖网络）
- **电机响应:** <100ms

### 资源使用
- **内存占用:** ~60% SRAM（约300KB）
- **Flash使用:** ~1.6MB应用程序 + 320KB文件系统
- **CPU使用率:** 平均65%（峰值可达90%）
- **网络流量:** 50-100KB/次AI分析

### 并发能力
- **HTTP连接:** 最多7个并发连接
- **FreeRTOS任务:** 5-8个活跃任务
- **API处理:** 10-20 req/s
- **WebSocket:** 支持多客户端实时通信
